# Intro
I used many online resources to build this project, the main goal was ultimately to learn at an extremly deep level how compilers work, and in particular how the LLVM toolchain worked. For this reason I used mostly online resources, rather than LLMs such as ChatGPT.
# Books

- Writing a C compiler; Nora Sandler: https://github.com/VadymMakohon/eBook/blob/main/Sandler%20Nora%20-%20Writing%20a%20C%20Compiler%20-%202024.pdf
